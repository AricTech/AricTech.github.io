About The Projects

Each Project I do I post on this website to document and explain how I did it. Please feel free to let me know if you see something I did wrong or something I could do better! Also let me know if you think of a good project you think I should try. Feedback is always welcome (as soon as I get the feedback page working...)! I will list all the projects I have done or I am planning on doing with a quick description below.

This Website!

I eventually hope to use this website to advertise my (not yet created) business but for now it is just showcaseing the projects I am working on and what I have worked on. Right now this website is in development! I am currently building the website on my ubuntu desktop using apache2 but the site will moved to a VM. This way I can back up the server easily and reduce the footprint. I will use Nginx to reverse proxy to all of the services because I only have one ip address.

Pi-Hole

The Pi-Hole is a dns server that blocks the dns records for known add related IPs. I will also try to use this as the DNS server for my local network.

The following projects have not been started yet.

Nginx

I plan on using this to reverse proxy my public ip address because I only have one public ip.

Fail2Ban

A security tool that will block repeated attempts to ssh into our network. I may or may not need this depending on if i have a public ssh server. I will most likely use openvpn in its place.

Password Vault

This is going to be my hardest project. I want to program my own password vault. I will create a server that stores the passwords in an encrypted sql database. I will then create a client that can add and edit the database files ment for that user. The client will be a phone app.
